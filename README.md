# second-mind
🧠 Interactive Digital Second Brain with neural graph visualization, note management, categories, search, dark mode, and local storage.
Neural Second Brain

Neural Second Brain is a personal knowledge management system inspired by modern "Second Brain" concepts. Instead of traditional note-taking, information is visualized as interconnected nodes inside a dynamic neural network.

Features
Interactive Graph-Based Notes
Create, Edit & Delete Nodes
Category Clustering
Search System
Dark & Light Mode
Local Storage Persistence
Drag-and-Drop Node Management
Knowledge Visualization
Responsive Design
Built With
HTML5
CSS3
JavaScript (Vanilla)
